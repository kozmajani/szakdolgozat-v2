## Dependencies
```npm install ffmpeg```
```npm install fluent-ffmpeg```
```npm install @material-ui/core```
```npm install @material-ui/core```



### To Run it:
>```cd server```
>
>```npm install```
>
>```npm run serve```

>Backend starts at port: 9000
>

>
>```cd client```
>
>```npm install```
>
>```npm start```

>Frontend starts at port: 3000
